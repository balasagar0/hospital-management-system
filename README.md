# Hospital Management System

A simple hospital management system with Spring Boot backend and React frontend.

## Features
- User authentication (login/signup)
- Hospital dashboard
- REST API endpoints

## Setup

### Backend
```bash
cd hospital-backend-jenkins
mvn spring-boot:run
```

### Frontend
```bash
cd hospital-frontend
npm install
npm run dev
```

## API Endpoints
- POST /auth/signup
- POST /auth/login

## Technologies Used
- Spring Boot
- React
- H2 Database
- Maven
